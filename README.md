🌍 Sasta Search Engine

A Python-based web scraping project that extracts country information directly from Wikipedia using BeautifulSoup and Requests.

This program allows users to search for any country and automatically fetches a short description from its Wikipedia page.

📌 Project Overview

This script:

✅ Scrapes the list of countries by area from Wikipedia
✅ Extracts all country page links
✅ Accepts user input for country search
✅ Fetches country information dynamically
✅ Displays country description with source URL

The project demonstrates practical usage of:

Web Scraping

HTML Parsing

Data Cleaning

User Input Handling

Automation using Python
